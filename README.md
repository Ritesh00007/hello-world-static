# 🌍 hello-world-static

> The simplest possible single-page website — a clean HTML page that says "Hello World" with a centered heading and a nice background color.

## ✨ Features

- 👋 Displays a bold "Hello World" heading
- 🎨 Clean, centered layout with a pleasant background color
- ⚡ Zero dependencies, zero build steps, zero configuration
- 📄 Single self-contained HTML file

## 🚀 Getting Started

### Prerequisites

- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No package manager, runtime, or server required

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/hello-world-static.git
cd hello-world-static
```

No dependencies to install:

```bash
echo 'No dependencies to install'
```

### Running Locally

Simply open the HTML file in your browser:

```bash
open index.html
```

Or on Linux:

```bash
xdg-open index.html
```

Or on Windows:

```bash
start index.html
```

That's it — the page will load instantly in your default browser. ✅
